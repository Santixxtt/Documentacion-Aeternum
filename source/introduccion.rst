Introducción
============

**Aeternum** es una plataforma de biblioteca virtual que permite:

- Registro e inicio de sesión seguro
- Préstamos físicos y digitales
- Lectura en línea
- Lista de deseos
- Sistema de reseñas y calificaciones
- Gestión por parte de bibliotecarios

Tecnologías principales:

- **Frontend:** React + Vite
- **Backend:** FastAPI (Python)
- **Base de datos:** MySQL / Railway
- **Cache y sistema de bloqueo:** Redis
- **Autenticación:** JWT
